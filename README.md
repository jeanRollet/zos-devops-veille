# z/OS – AWS – DevOps – Veille & Parcours Techniques

Ce dépôt rassemble mon travail d'exploration, de documentation et de démonstration des pratiques DevOps appliquées aux plateformes mainframe, en particulier :

- IBM z/OS (via Hercules 4.9 et z/OS 3.1 VSI)
- IBM i / AS400 (à venir)
- AWS EC2 pour la simulation
- Automatisation (systemd, scripts)
- Zowe CLI, Git, pipelines CI/CD
- Analyse d’architectures Legacy (IMS, CICS, DB2, batch)

---

## Structure du dépôt

- `fiches/` : fiches synthétiques (états stables, reproductibles)
- `parcours/` : notes détaillées de travail et d'expérimentation
- `raw/` : logs, configs, dumps, données brutes
- `templates/` : modèles de fiches
- `scripts/` : scripts d’automatisation documentaire
- `urls-to-process.md` : liens trouvé à analyser
- `index.md` : index général

---

## Objectifs

1. **Montrer concrètement** la faisabilité DevOps sur z/OS et iSeries.  
2. **Rendre visible le Legacy** : architecture, batchs, transactions, données.  
3. **Préparer des scénarios de modernisation** (cloud, portage, replatforming).
4. **Créer une base documentaire vivante**, comme alternative aux PDF obsolètes.
